# 🏛️ The Obsidian Palace: Admin Command Guide

Welcome to the inner sanctum of **DINA COSMETIC**. This guide provides the protocols for managing your luxury empire.

---

## 🏗️ 1. Inventory Management (The Vault)
**Location:** `Admin Portal > Products`

### Forge New Item
- **Name & Description:** Use high-end, evocative language.
- **Base Price:** The entry point for the acquisition.
- **Images:** Upload artifacts. The first image will be the primary manifestation.
- **Variants:** Essential for stock. Each variant requires a **Quantity** to be manifestable in the store.

### Updating Artifacts
- Use the **Pencil icon** to re-forge existing items.
- Use the **Trash icon** to banish an item to the void. *Warning: Permanent action.*

---

## 📦 2. Acquisitions & Logistics (Orders)
**Location:** `Admin Portal > Orders`

### Order Protocol
1. **Paid Status:** When a client completes a ritual at checkout, the order appears as **PAID**.
2. **Fulfillment:** Click **"Fulfill Order"** to:
   - Generate a **Shippo Manifest** (Shipping Label).
   - Trace the tracking number.
   - Dispatch an **Obsidian Notification** (Email) to the client.
3. **Tracking:** Use the **External Link** icon to view the generated shipping manifest.

---

## 👥 3. Palace Residents (Customers)
**Location:** `Admin Portal > Customers`

- Monitor the elite clientele who have entered The Palace.
- View arrival dates (joined at) and email coordinates.
- **Search:** Use the search bar for direct locator services.

---

## ⚙️ 4. Palace Configuration (Settings)
**Location:** `Admin Portal > Settings`

- Initialize core systems.
- Sync logistical coordinates.

---

## 🛡️ Administrative Security
- Access is restricted to accounts with **Admin Clearance** in the Supabase `profiles` table.
- All rituals are performed at the **Edge** via unified security protocols (`proxy.ts`).
